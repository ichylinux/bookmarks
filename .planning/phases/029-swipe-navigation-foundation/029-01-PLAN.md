  <action>
Inside the `$(function() { ... })` block, after the click handler, add a `document.querySelectorAll('.portal').forEach(function(portalEl) { ... })` block that attaches native touch listeners to each `.portal` element.

**Critical**: Do NOT use jQuery event delegation on `$document` for touch events. RESEARCH.md §6 requires native `addEventListener` with `{ passive: false }` on `touchmove` so that `e.preventDefault()` is not silently ignored by Chrome.

Declare state variables per portal element inside the forEach callback closure:
```javascript
let startX = 0, startY = 0, scrollIntent = false, totalDx = 0;
```

**touchstart** (`{ passive: true }`):
- Read `e.touches[0]` and record `startX = t.clientX`, `startY = t.clientY`
- Reset `scrollIntent = false`, `totalDx = 0`

**touchmove** (`{ passive: false }`):
- Return immediately if `scrollIntent` is true
- Read `t = e.touches[0]`; compute `dx = t.clientX - startX`, `dy = t.clientY - startY`
- If `Math.abs(dx) + Math.abs(dy) < 10`: return (below 10px intent threshold — no decision yet)
- If `Math.abs(dy) > Math.abs(dx)`: set `scrollIntent = true`; return (vertical intent — SWIPE-02 path)
- Call `e.preventDefault()` (suppress browser scroll for horizontal gesture)
- Set `totalDx = dx`

**touchend** (no passive option needed):
- Return if `scrollIntent` is true or `Math.abs(totalDx) < 50` (gesture below threshold)
- `const $portal = $(portalEl)`
- `const $tabs = $portal.prev('.portal-column-tabs')`; return if `!$tabs.length`
- `const $activeTab = $tabs.find('.portal-column-tab--active')`
- `const currentIndex = parseInt($activeTab.attr('data-portal-column-index'), 10)`; return if `Number.isNaN(currentIndex)`
- `const colCount = $tabs.find('.portal-column-tab').length`
- Direction convention (standard mobile UX): `totalDx < 0` (left swipe) → next column (+1); `totalDx > 0` (right swipe) → previous column (-1)
- `const direction = totalDx < 0 ? 1 : -1`
- `const newIndex = Math.min(Math.max(currentIndex + direction, 0), colCount - 1)` (clamp — SWIPE-03)
- If `newIndex !== currentIndex`: call `activateColumn($portal, $tabs, newIndex)`
