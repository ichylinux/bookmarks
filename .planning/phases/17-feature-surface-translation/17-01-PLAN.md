<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Locale catalog -> rendered UI | Static translations become user-visible HTML and JavaScript-fed attributes in later plans. |
| User/external content -> translated sentence interpolation | Bookmark titles, note bodies, todo titles, feed titles, calendar titles, and holiday names may be interpolated but must not become translation keys. |
| Request locale -> model helper output | Request-scoped `I18n.locale` controls plain Ruby model/gadget helper output. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-17-01-04 | E | Locale selection | accept | Phase 14 already enforces supported locales before `I18n.with_locale`; this plan does not add new locale input channels. |
