Create NEW file `app/controllers/notes_controller.rb` with the following specification (implement directly — wording may match exactly):

```ruby
class NotesController &lt; ApplicationController
  def create
    @note = Note.new(note_params)

    if @note.save
      redirect_to root_path(tab: 'notes')
    else
      redirect_to root_path(tab: 'notes'), alert: @note.errors.full_messages.to_sentence.presence || 'エラーが発生しました'
    end
  end

  private

  def note_params
    params.require(:note).permit(:body).merge(user_id: current_user.id)
  end
end
```

Mandatory rules:
1. **`user_id`** MUST NEVER appear inside `permit(...)` — only merged from `current_user.id` after permit.
2. **Success redirect:** MUST be exactly `redirect_to root_path(tab: 'notes')` — no keyword flash required on success unless you voluntarily add notice (omit for parity with roadmap SC#1 as written).
3. **Failure redirect:** MUST use `redirect_to root_path(tab: 'notes'), alert: ...` (CONTEXT D-01 / D-02). Use **`@note.errors.full_messages.to_sentence`** so Japanese locale (`config.i18n.default_locale = :ja`) yields readable copy — falls back literal only if Rails returns an empty sentence.
4. **Do NOT** define `destroy` or other actions — out of routed scope after Task 1.
