# Quick Task 260501-q05: Font Size Preference

Support a persisted user font size preference. Users can choose one of three sizes from preferences: large, medium, or small.

## Tasks

1. Add persisted preference
   - Add nullable `preferences.font_size` with no database default.
   - Add model constants/options and inclusion validation for `large`, `medium`, `small`, while allowing nil.
   - Fall back to `"medium"` at render time when the stored preference is nil.
   - Update test helper and model tests.

2. Expose in preferences UI
   - Permit `font_size` in nested preference attributes.
   - Add a Japanese-labeled select to the existing preferences form.
   - Cover saving and form rendering in controller tests.

3. Apply the preference
   - Add helper support for a safe font-size body class.
   - Render both theme and font-size classes on `<body>`.
   - Add CSS for `body.font-size-small`, `body.font-size-medium`, and `body.font-size-large`.
   - Cover rendered body class in tests.

## Verification

Run:

```bash
bin/rails db:migrate RAILS_ENV=test
bin/rails test test/models/preference_test.rb test/controllers/preferences_controller_test.rb
```
