# Quick Task 260429-q02: Rename デフォルト theme to クラシック, make モダン the default

## Goal

In the preferences theme select:
1. Rename the blank option 'デフォルト' to 'クラシック'
2. Make 'モダン' appear first (visually the default)

## Tasks

### Task 1: Update theme select in preferences view

**File:** `app/views/preferences/index.html.erb`

Change:
```ruby
f.select :theme, {'シンプル' => 'simple', 'モダン' => 'modern'}, include_blank: 'デフォルト'
```

To:
```ruby
f.select :theme, {'モダン' => 'modern', 'シンプル' => 'simple', 'クラシック' => ''}
```

- 'モダン' first = visually the default
- 'クラシック' replaces 'デフォルト' as the empty-value option (classic look, no CSS class)
- `default_preference` already sets `theme = "modern"` ✓

## Verify

- Preferences page renders with 'モダン' as first option
- Option previously labeled 'デフォルト' is now labeled 'クラシック'
- Existing users with theme='' see 'クラシック' selected
- Tests pass
