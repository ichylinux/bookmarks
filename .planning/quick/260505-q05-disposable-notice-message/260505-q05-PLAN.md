# Quick Task Q-05: Disposable Notice Message

## Objective

Allow users to dismiss the login success notice, including "Google アカウントによる認証に成功しました。", without refreshing the page.

## Tasks

1. Render shared flash notice/alert messages with an accessible close button while preserving existing flash text output.
2. Add JavaScript that removes the clicked flash message from the DOM.
3. Add focused layout and JavaScript contract tests, resolve any unrelated E2E selector ambiguity blocking the suite, then run lint and relevant Rails tests.
