# Quick Task 260504-q02: HolidayJp Holidays Only In Japanese Locale

## Summary

Japanese holidays from `holiday_jp` are now gated by the active locale. Calendars only mark and render HolidayJp holidays when `I18n.locale` is `:ja`; English and other locales return no HolidayJp holiday state or name.

## Changed

- Updated `Calendar#holiday?` and `Calendar#holiday` to return no holiday unless the active locale is Japanese.
- Updated model coverage for Japanese and English holiday behavior.
- Updated calendar gadget controller coverage so English locale does not render `元日`.
